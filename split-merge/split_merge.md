# 스플릿 머지 정렬

## 정렬 방법

길이가 $N$인 수열 $A=[A_1,A_2,⋯,A_N]$에 대하여, $A$가 정렬될 때까지 다음 과정 $X$를 반복한다.

1. 과정 $X$의 실행 횟수가 최소회되는 양의 정수 $x$를 구한다.
2. 수열 $A$의 원소 중 값이 $x$ 이하인 원소들을 원래 순서대로 추출하여 부분 수열 $B$를 만든다.
3. 수열 $A$의 원소 중 값이 $x$ 초과인 원소들을 원래 순서대로 추출하여 부분 수열 $C$를 만든다.
4. 기존 수열 $A$를, $B$와 $C$를 순서대로 이어 붙인 수열($B+C$)로 대체한다.

## 코드 설명

- `get_x_count.py` `get_x_count.cpp` - 과정 $X$의 최소 실행 횟수를 구하는 코드이다.
